# Ne Jowo (.nj) Programming Language

Bahasa pemrograman kustom berbasis Python dengan cita rasa kearifan lokal (Bahasa Jawa) yang dirancang khusus untuk dijalankan di lingkungan CLI/Terminal seperti Termux.

## 🚀 Fitur Utama
- **`aturaken`** : Perintah mencetak teks/variabel ke layar (`print`).
- **`gadhah`** : Operator memasukkan nilai/data ke variabel (`=`).
- **`nyuwun`** : Meminta input interaktif dari pengguna (`input`).
- **`yen ... banjur ... utawa`** : Logika pengkondisian (`if-else`).
- **`mbaleni`** : Fitur perulangan (`while loop`).
- **Matematika Dasar** : Mendukung `ditambah`, `dikurangi`, `diping` (perkalian), dan `diporo` (pembagian).
- **Ekstensi File** : Menggunakan berkas berekstensi khusus `.nj`.
